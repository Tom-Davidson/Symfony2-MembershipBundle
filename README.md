Symfony2-MembershipBundle
=========================

A bundle for managing a membership list.

Installation
============
to composer.json add:
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/Tom-Davidson/Symfony2-MembershipBundle"
        }
    ],

and to the require:
	"tomdavidson/membershipbundle": "*"
